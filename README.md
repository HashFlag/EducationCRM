# EducationCRM-python

#### 介绍
CRM管理系统,可自定制表单展示

#### 软件架构
Django框架，mysql数据库，还有部分插件，python3.7，bootstrap3模板使用


#### 安装教程

1.  安装python3.7
2.  安装pycharm
3.  安装Django框架
4.  安装mysql数据库
5.  导入项目教程，请看私人博客：https://www.cnblogs.com/wylshkjj/p/11983596.html
